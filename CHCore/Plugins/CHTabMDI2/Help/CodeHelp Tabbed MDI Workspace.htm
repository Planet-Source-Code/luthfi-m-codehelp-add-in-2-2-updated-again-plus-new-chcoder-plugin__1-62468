CodeHelp Tabbed MDI Workspace

 

There are a few options available for this plug-in, as it designed with simplicity in mind.

Most of the features are accessible by right clicking on the tab panel area.

And all the function should be self-explanatory as in the screenshot below.

 

 

Here is the list of functionalities of CHTabMDI:

  • Close any visible tab (active or not) by right-clicking on desirable tab and choose ‘Close’ menu item.
  • Close all opened documents
  • Close all opened documents except for the currently active document.
  • Select from 3 ways of displaying close icon for the tabs which are:

o       Hidden (the close icon is not displayed)

o       Rightmost (the close icon is showed on the right most of the tab area)

o       On Active Tab (the close icon is showed only on the active tab)

 

  • The tab also accessible with keyboard shortcut, you can press CTRL + (number from 0 to 9) to activate any visible tab without using the mouse.

The plug-in will show on-screen hint whenever CTRL key is pressed, so you’ll know what number corresponds to each tab.
